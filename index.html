Getting started #back to top

Firstly, we must create a class inheriting "Engine" class:

//Header file
#pragma once
#include "Engine.h"

using namespace neno;

class GameEngine :
    public Engine
{
    void Update(float framesPerSecond);
    void Render();
    void Resize(int width, int height);
};

//Source
#include "GameEngine.h"

void GameEngine::Update(float framesPerSecond)
{

}

void GameEngine::Render()
{

}

void GameEngine::Resize(int width, int height)
{

}

In Update method, we can put all of game logic, like hero movement, keyboard & mouse & gamepad support.

Render method is place for all of drawing.

Resize method is called when we changed window size

Keyboard & Mouse support #back to top

Neno engine contains several input functions.

                                //Source
                                #include "GameEngine.h"
                                #include "Neno.h"
                                
                                void GameEngine::Update(float framesPerSecond)
                                {
                                    if (Keyboard::IsPressed('a')) //Check button is pressed
                                        hero_x -= 5;
    
                                    Keyboard::IsDown(NENO_KEY_PAGEUP); //Check button is held.
                                    Keyboard::IsUp(key); //Check button not pressed
                                    Keyboard::IsReleased(key); //Check button is released
                                    //In "key" parameters we can use Pre-defined const like NENO_KEY_PAGEUP or simply char. See Keyboard.h for more
                                    Keyboard::IsShift(); //Check shift held (or CAPS LOCK without shift)
                                    Keyboard::IsCtrl(); //Check ctrl held
                                    Keyboard::IsAlt(); //Check alt held
    
                                    Mouse::GetX(); //Get mouse x relative to window
                                    Mouse::GetY(); //Get mouse y relative to window
                                    Mouse::LeftPressed(); //Check left mouse button pressed
                                    Mouse::RightPressed(); //Check right mouse button pressed
                                    Mouse::MiddlePressed(); //Check middle mouse button pressed
                                }

Textures & Sprites #back to top

Texture is a simple class used for storage & drawing. Instead of Texture class, Sprite class is a complex version of Texture. Sprite can rotate, scale, change origin, define x and y, and storage it for later use

                                //Source
                                #include "GameEngine.h"
                                #include "Neno.h"
                                
                                Texture* texture;
                                Sprite* sprite;
                                
                                GameEngine::GameEngine()
                                {
                                    texture = new Texture(Utils::FilePath("Content/image.png")); //Load image from file
                                    sprite = new Sprite(texture, 0, 0, 16, 16, 8, 8, 1.0, 1.0, 45.0, &Color::Aquamarine);
                                }
                                
                                void GameEngine::Render()
                                {
                                    texture->Draw(32, 32, 64, 64, Color::Red);
                                    sprite->Draw();
                                }

In this example, we created Sprite from existing Texture, place it at [0;0], with size [16;16], with origin (offset relative to base sprite position) [8;8], scale [1;1], angle 45 degrees and color Aquamarine (Color class contains a lot of pre-defined colors). If we want to draw Texture, we must pass some parameters (described from parameter names). If we want to draw Sprite, we must simply call Draw method.
